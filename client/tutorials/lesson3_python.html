This tutorial deals with basic operations on integers, floating point values, and strings.

All basic arithmetic operations(+ , - , *, /) can be performed on integers and floating point values. We can also perform other operations such as the remainder operator(%) on these types. As an exercise, use basic operators and dsiplay the result.

If we perform operations between integers and floating point values, the result is automatically converted to a float. This is known as implicit type conversion.

Type conversion can also be done by stating the new type in parentheses. This is called explicit type conversion. For an example, look at the provided starter code.
